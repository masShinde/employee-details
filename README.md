# Employee Details

A React application to manage and view employee information.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
git clone https://github.com/yourusername/employee-details.git
cd employee-details
npm install
```

### Running the App

```bash
npm start
```

The app will be available at `http://localhost:5173`.
